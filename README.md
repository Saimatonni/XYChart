# XYChart

First have to install the relevant packages:

### `npm install --save @visx/mock-data @visx/group @visx/shape @visx/scale`

## Available Scripts

In the project directory, you can run:

### `npm start`

### `npm test`

### `npm run build`

### `npm run eject`
